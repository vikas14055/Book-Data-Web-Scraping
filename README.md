
# 📚 Book Data Web Scraping

## Overview
This project extracts book names and prices from the Books to Scrape website using Python.

## Technologies Used
- Python
- Requests
- BeautifulSoup
- Pandas

## Features
- Scrapes book titles
- Scrapes book prices
- Saves data into a CSV file

## Files
- webscrapping.py
- books_dataset.csv

## How to Run

```bash
pip install requests beautifulsoup4 pandas
python webscrapping.py
```


## Output

The program creates a CSV dataset containing:
- Book Name
- Price
